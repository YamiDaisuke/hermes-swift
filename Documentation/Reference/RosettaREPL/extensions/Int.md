## Methods
### `%%(_:_:)`

```swift
static func %% (lhs: Int, rhs: Int) -> Int
```

Execute a real module operation, supporting negative and positve values
- Parameters:
  - lhs: The dividend
  - rhs: The divisor
- Returns: The unsigned reminder of the division
